SCENARIO 2 — Search and Filter
================================
Steps:
1. Click Assessments in navbar
2. Type slowly in search box
3. Say: "Notice it waits 400ms
         before calling API —
         this is debounce to avoid
         hammering the server"
4. Select status from dropdown
5. Click Export CSV button

Expected Output:
- Results filter as you type
- CSV file downloads

SCENARIO 3 — Security Demo
============================
Steps:
1. Open Postman or browser
2. Call API without token:
   GET http://localhost:8080/api/assessments/all
3. Show 401 Unauthorized response
4. Say: "Without JWT token
         all endpoints return 401.
         Spring Security protects
         every route."

Expected Output:
- 401 Unauthorized response
- Shows security working

TALKING POINTS:
===============
Q: What does it do?
A: Privacy Impact Assessment Tool helps
   organizations evaluate how their
   projects handle personal data.

Q: What AI is used?
A: Groq API with LLaMA-3.3-70b model
   via Flask microservice on port 5000.
   It describes assessments, recommends
   actions and generates reports.

Q: What security measures?
A: JWT authentication, Spring Security,
   rate limiting, input sanitization,
   SQL injection protection.
